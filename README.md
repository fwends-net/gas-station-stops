## The Solution, for Real

Plan your ride around gas stations at the right locations. With the Gas Station Planner, you upload your GPX file and get a list of gas stations that are on the route or reachable via a small detour. Pick your favorite and mark it on your ride, or replan the route to pass the station.

## Creation

This tool was vibe coded in 10 minutes and is yet another example of the capabilities of AI coding assistants.
The human part of this creation was the expression of a need and interest: I simply said Claude Code that I wanted a tool that gave me all gas stations along a planned route.
After it built a Python script that did exactly that, I understood that it was possible, and asked it to build an interactive web tool for it.
